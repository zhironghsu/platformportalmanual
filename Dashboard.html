Dashboard

Dashboard

The Dashboard provides a summary of system information and a WISE-Edge365, (aka DeviceOn/BI) widget.

WISE-Edge365 setting

The WISE-Edge365 setting widget is the same in the Setting page.

System Information

Device Name from the BIOS.

OS Information

OS type (Windows, Linux), architecture (x86, x86_64, armv7l, arm64), and the Embedded Image version (if available).

CPU Information and Usage Monitoring

Monitoring Temperature, Voltage, Current, and FAN speed.
