ETP

ETP

GET /platform/api/etp/devicedata

Get ETP device data.

  • Request Headers

    Login with OS account to get JWT token (token field from the response body).

    Authorization: Bearer <token>

  • Response 200 OK

    {
    "modelName": "XXXXX",
    "serialNumber": "Default string",
    "boardVersion": "Default string",
    "uuid": "00020003-0004-0005-0006-000700080009",
    "version": ""
}

  • Response 401 Unauthorized

    {
    "Code": 401,
    "Message": "<the detail error message>"
}

  • Response 204 No Content

    Not supported or no available data.

GET /platform/api/etp/userdata

Get ETP user's data.

  • Request Headers

    Login with OS account to get JWT token (token field from the response body).

    Authorization: Bearer <token>

  • Response 200 OK

    {
    "data": {
        "userSpace1": "<base64 string>",
        "userSpace2": "<base64 string>"
    },
    "locked": true | false
}

  • Response 400 Bad Request

    The request has applied, but got errors.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

  • Response 401 Unauthorized

    {
    "Code": 401,
    "Message": "<the detail error message>"
}

  • Response 404 Not Found

    Not supported or no available data.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

PUT /platform/api/etp/userdata

Set ETP user's data.

  • Request Headers

    Login with OS account to get JWT token (token field from the response body).

    Authorization: Bearer <token>

  • Request Body

    The maximum length of userSpace1 and userSpace2 is 128 bytes. (RAW data)

    {
  "userSpace1": "<string | base64 string>",
  "userSpace2": "<string | base64 string>"
}

  • Response 200 OK

    {
  "userSpace1": "<same as request body>",
  "userSpace2": "<same as request body>"
}

  • Response 400 Bad Request

    The request has applied, but got errors.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

  • Response 401 Unauthorized

    {
    "Code": 401,
    "Message": "<the detail error message>"
}

  • Response 404 Not Found

    Not supported.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

  • Response 422 Unprocessable Entity

    The request body has missing parameters.

    {
    "errors": {
        "validator": "<the detail validator error>"
    }
}

    The userSpace1 or userSpace2 raw data length is out of range (128 bytes).

    {
    "errors": {
        "ETP": "The maximum length of UserSpace1 is 128"
    }
}

PUT /platform/api/etp/userdata/lockstatus

Lock or unlock the ETP user's data area.

  • Request Headers

    Login with OS account to get JWT token (token field from the response body).

    Authorization: Bearer <token>

  • Request Body

    The maximum length of password is 8 byre.

    {
  "locked": true | false,
  "password": "<maximum 8 bytes password>"
}

  • Response 200 OK

    {
  "locked": true | false, // same as request body
}

  • Response 400 Bad Request

    The request has applied, but got errors.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

  • Response 401 Unauthorized

    {
    "Code": 401,
    "Message": "<the detail error message>"
}

  • Response 404 Not Found

    Not supported.

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}

  • Response 422 Unprocessable Entity

    The request body has missing parameters.

    {
    "errors": {
        "validator": "<the detail validator error>"
    }
}

    The password length is out of range (1~8 bytes).

    {
    "errors": {
        "ETP": "<the detail error message>"
    }
}
